# SQLite3 ActiveRecord REGEXP
This adds REGEXP support to SQLite3 in ActiveRecord.

## Installation
    $ gem install sqlite3_ar_regexp

If you're using Bundler, add this to your Gemfile.

    gem 'sqlite3_ar_regexp'
